# homebrew-tap

Homebrew tap for [say8425](https://github.com/say8425)'s tools.

## Install

```bash
brew tap say8425/tap
brew install hs-cli
```

Or in one shot:

```bash
brew install say8425/tap/hs-cli
```

## Formulae

| Formula | Description | Upstream |
|---|---|---|
| `hs-cli` | Hearthstone CLI for AI agents and humans | [say8425/hs-cli](https://github.com/say8425/hs-cli) |

## Updating

Formulae here track GitHub Releases of the upstream repos. When a new version is published, this tap is updated either manually or by an automation workflow in the upstream repository.
